Cafe Lestari System
Cafe Lestari System is a digital platform designed to streamline cafe operations, including order management, menu control, and sales tracking. Built with web technologies, it facilitates efficient, secure administrative tasks for staff. Read the full documentation or view the codebase directly at GitHub.
________________________________________
📌 Features
•	Order Management: Process customer receipts, manage tables, and track active billing.
•	Menu Control: Update item availability, edit pricing, and organize culinary categories.
•	Sales Analytics: Generate financial reports, track peak hours, and export revenue summaries.
•	User Authentication: Secure role-based access for system administrators, cashiers, and kitchen managers.
________________________________________
💻 Tech Stack
•	Frontend: HTML5, CSS3, JavaScript
•	Backend: PHP
•	Database: MySQL
________________________________________
🚀 Getting Started
Follow these steps to deploy the system locally.
Prerequisites
•	Install a local web server environment such as XAMPP or WampServer.
•	Ensure you have Git installed on your local machine.
Installation
1.	Clone the repository
bash
git clone https://github.com
cd Cafe-Lestari-System
Use code with caution.
2.	Move files to the local server directory
o	Cut and paste the repository folder into your server's root web directory (e.g., C:/xampp/htdocs/ for Windows XAMPP users).
3.	Set up the database
o	Open your web browser and go to your server management panel, typically at http://localhost/phpmyadmin.
o	Create a new database named cafe_lestari.
o	Import the provided SQL database template file (look for a .sql file inside the repository directory).
4.	Configure database credentials
o	Open the configuration file (usually named config.php, database.php, or connection.php) in a text editor.
o	Update the database name, host user (root), and password fields to match your local database settings.
5.	Run the application
o	Navigate to http://localhost/Cafe-Lestari-System in your browser.
